# GraphQL Full-Stack Example (React + Apollo + Express)

This repository contains a **simple full-stack GraphQL setup**:

- **Backend**: Express + Apollo Server (GraphQL API)
- **Frontend**: React (Vite) + Apollo Client
- Data source: `jsonplaceholder.typicode.com` (mock REST API)

---

## Project Structure

```
root/
├── server/        # GraphQL backend (Express + Apollo Server)
└── client/        # React frontend (Vite + Apollo Client)
```

---

## Prerequisites

- Node.js >= 18
- npm

---

## Backend (GraphQL Server)

### Setup

```bash
cd server
npm install
```

### Run server

```bash
npm start
```

Server will start at:

```
http://localhost:8000/graphql
```

### What it does

- Exposes GraphQL **Queries & Mutations**
- Fetches data from external REST APIs using `axios`
- Example entities: `User`, `Todo`

---

## Frontend (React Client)

### Setup

```bash
cd client
npm install
```

### Run client

```bash
npm run dev
```

App will be available at:

```
http://localhost:5173
```

---

## Apollo Client Setup

- Apollo Client is configured in `src/apolloClient.js`
- Wrapped using `ApolloProvider` in `main.jsx`
- Uses `useQuery` and `useMutation` for GraphQL operations

---

## Example GraphQL Query

```graphql
query GetUser($id: ID!) {
  getUserById(id: $id) {
    name
    email
    todos {
      title
      completed
    }
  }
}
```

### Variables

```json
{
  "id": "1"
}
```
